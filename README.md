### SvelteKit TypeScript Static Boilerplate

Minimalistic boilerplate for SvelteKit projects in TypeScript, optimized for SSG.

#### Prerequisites

- [Node.js](https://nodejs.org/) (^18.18)
- [pnpm](https://pnpm.io/) (^8.15)

#### Getting Started

1. Clone repo:

      ```bash
      git clone https://github.com/equilpres/sveltekit-typescript-static-boilerplate.git
      ```

2. Go to project dir:

      ```bash
      cd sveltekit-typescript-static-boilerplate
      ```

3. Install deps:

      ```bash
      pnpm install
      ```

#### Scripts

- `dev`: Start dev server.

     ```bash
     pnpm dev
     ```

- `build`: Build project.

     ```bash
     pnpm build
     ```

- `preview`: Preview prod build.

     ```bash
     pnpm preview
     ```

- `sync`: Sync files with SvelteKit.

     ```bash
     pnpm sync
     ```

- `svelte-check`: Check Svelte files.

     ```bash
     pnpm svelte-check
     ```

- `lint`: Lint code.

     ```bash
     pnpm lint
     ```

- `fmt`: Format code.

     ```bash
     pnpm fmt
     ```

- `bump`: Update deps.

     ```bash
     pnpm bump
     ```
